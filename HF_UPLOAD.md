# Hugging Face Dataset Upload Notes

Dataset package path:

```bash
datasets/atu-trace-1000
```

HF-ready mirror path generated by `make publication-bundles`:

```bash
hf_dataset/atu_trace_1000
```

Suggested upload target:

```text
joy7758/atu-trace-1000
```

Authentication boundary:

No Hugging Face upload is claimed unless `huggingface-cli whoami` succeeds and
the dataset repository is created or updated on the Hugging Face Hub.

CLI path:

```bash
.venv/bin/python -m pip install -U huggingface_hub datasets
.venv/bin/hf auth login
.venv/bin/hf upload joy7758/atu-trace-1000 hf_dataset/atu_trace_1000 . --repo-type dataset --commit-message "ATU v0.2 dataset initial release"
```

Python upload example after authentication:

```python
from huggingface_hub import HfApi

api = HfApi()
api.create_repo(
    repo_id="joy7758/atu-trace-1000",
    repo_type="dataset",
    exist_ok=True,
)
api.upload_folder(
    repo_id="joy7758/atu-trace-1000",
    repo_type="dataset",
    folder_path="hf_dataset/atu_trace_1000",
)
```
